# 🧭 Navbar Animada

> Uma barra de navegação interativa com animações suaves entre os itens do menu.

---

## 📋 Sobre o Projeto

A **Navbar Animada** é uma barra de navegação moderna e responsiva, com transições e efeitos de animação ao interagir com os itens do menu, destacando visualmente a opção selecionada.

---

## 🚀 Funcionalidades

- Menu de navegação com ícones e textos
- Item ativo destacado (fundo colorido + texto em branco)
- Animação de transição ao trocar de item selecionado
- Itens do menu:
  - 🏠 Home
  - 📰 Blog
  - 💻 Code
  - ✉️ Help
  - 👤 About

---

## 🗂️ Estrutura do Projeto

```
NAVBAR ANIMADA/
├── index.html     # Estrutura da página
└── style.css      # Estilização e animações
```

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura da página
- **CSS3** — Estilização, transições e animações

---

## ▶️ Como Executar

1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` diretamente no navegador

Ou use uma extensão como **Live Server** no VS Code para rodar em `http://127.0.0.1:5500`.

---

## 📐 Como Funciona

1. Ao clicar em um dos itens do menu, ele se torna o item **ativo**
2. O fundo colorido se move suavemente até o item selecionado (efeito de animação)
3. O texto do item ativo muda de cor para contraste
